함수

* 선언적 함수

      function 함수명([매개변수, 매개변수, ...]) {
         함수호출 시 실행할 코드;
         [return 결과값;]
      }

* 익명 함수(비선언적 함수)

      변수|속성 = function([매개변수, 매개변수, ...]) {
         실행할 코드;
         [return 결과값;]
      }
- 특정 변수나 속성에 대입되는 함수 제시시 주로 사용됨 (주로 이벤트 핸들러 작성시 사용)

* 화살표 함수

      - 본문에 리턴만 있을 때
        변수 = ([매개변수, 매개변수, ...]) => 리턴값;
      - 본문에 내용이 여러줄일 때
        변수 = ([매개변수, 매개변수, ...]) => {
                                                실행코드,
                                                실행코드,
                                                ...
                                                [return 결과값;]
                                             }
      - 익명 함수를 간단히 작성하는 방법(ES6(ES2015)에 추가)
