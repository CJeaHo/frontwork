CSS 기본 선택자

선택자란?

특정 html요소를 선택하고자할 때 사용하는 기능
해당 요소를 선택하여 원하는 "스타일"과 "기능"을 적용시킬 수 있음

1. 모든(전체) 선택자: *

현재 이 문서의 모든 요소를 선택하고자할 때 사용

	* {
		스타일속성: 값;
		스타일속성: 값;
		...
	}

2. 태그 선택자: 태그명

현재 문서상에 해당 태그들 다 선택하고자 할 때 사용

	태그명 {
		스타일 속성: 값;
		스타일 속성: 값;
		...
	}

3. 아이디 선택자: #아이디명

현재 문서상에 특정 html 요소 "하나만"을 선택하고자 할 때 사용
단, 해당 요소에 id속성을 이용하여 고유한 아이디값을 부여해야함
(여러개 넣는다고 오류가 안생기나 여러개를 넣을거면 차라리 class를 사용하자) 

	#아이디명 {
		스타일 속성: 값;
		스타일 속성: 값;
		...
	}
  1. html5
  2. css3
  3. javascript
  4. jquery

4. 클래스 선택자: .클래스명

해당 문서내에 내가 원하는 요소 "여러개"를 선택하고자 할 때

	.클래스명 {
		스타일 속성: 값;
		스타일 속성: 값;
		...
	}
