jQuery 개요

* jQuery란?

기존의 javascript 언어만으로 구현하기엔 복잡햇던 구문 작성들을
간소화하기 위해 '존 레식'에 의해 만들어짐
javascript 기반 라이브러리
즉, javascript 언어로 유용한 함수나 내용들이 이미 정의되어있는 모음집
자주 사용되는 코드를 모듈화하여 제공하는 형태
개발자가 처음부터 기능을 구현하는 대신에 이미 작성된 코드를 사용하여 효율적으로 작업할 수 있다

복잡한 javascript 구문을 보다 짧고 쉽게 개발 가능

> jQuery의 장점
1) DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있음
2) ajax(비동기식 통신), 이벤트 처리 등 폭 넓게 지원
3) jQuery와 관련된 확장형 플러그인, 오픈 api등을 지원
(차트, 슬라이드, ...등등)

* jquery구문 작성하기

jquery 구문으로 DOM요소를 다루는 구문을 기본적으로 작성하기 때문에
문서상에 요소들이 다 만들어지고나서 실행되어야 함

* 순수 javascript 
   window.onload = function() {
      실행코드;
   }

   javascript에서의 window.onload구문은
   해당 문서의 요소들이 모두 로드되고 뿐만 아니라 연동되어있는
   외부문서들도 다 가져온 뒤에 실행됨
   이 문서에서 여러번 작성해도 제일 하단에 있는 구문 하나만 실행됨
> jQuery에서는 3가지 구문으로 제공 
   1) jQuery(document).ready(function(){
         실행코드;
      });

   2) $(document).ready(function(){
         실행코드;
      });
   
   3) $(function(){
         실행코드;
      });
      
   해당 문서의 요소들만 다 로드되면 곧바로 실행됨
   여러번 기술가능함(제한없음) 순서대로 실행
