* DOM(Document Object Model)

HTML에 있는 각각의 요소들을 노드(node)라고 함
> 요소 노드(Element Node): 태그 그 자체만을 의미
> 텍스트 노드(Text Node): 태그 내에 기록되는 내용

텍스트 노드가 존재하는 요소(시작태그와 종료태그가 한 쌍으로 이루어져 있음): div, h~, li, p, a
텍스트 노드가 존재하지 않는 요소(시작태그만 이뤄져 있음): input, img, br, hr

* 노드(요소) 생성과 관련된 메서드

- 텍스트 노드가 존재하는 노드 생성(시작과 종료태그)

- 텍스트 노드가 존재하지 않는 노드 생성(시작태그만 존재)

* 노드(요소) 삭제와 관련된 메소드
