font-size : 글꼴의 크기를 변경할 때 사용

	선택자 {font-size: px|em|%;}

	상대 크기 
	em: 부모 요소의 크기를 기준으로 배수, 1배=1em=100% 1.5배=1.5em=150%
	rem: 최상위 부모 요소의 크기를 기준으로 배수
	% 기본 설정된 크기에서 상대적으로 크기 설정(초기설정100%)

	절대 크기 
	px, pt
  • 고정크기 px테스트
  • 가변크기 em테스트
  • 가변크기 %테스트

font-weight : 글꼴의 두께 지정

	선택자 {font-weight: normal|bold|bolder|lighter|100|400|900;}
  • 기본 굵기
  • 굵은 글꼴
  • 굵은 글꼴보다 더 굵게
  • 기본 굵기보다 더 가늘게

font-variant : 영문텍스트 문구를 작은 대문자로 변경할 때 사용

선택자 {font-variant: small-caps;}
